# D022-TiAl3 Shock Compression MD Simulations

This repository contains representative input scripts, potential files,
analysis scripts, and selected datasets associated with the manuscript:

"Atomic-scale shock response of D022-TiAl3 intermetallics:
effects of piston velocity, grain size and initial temperature"

## Contents

* LAMMPS input scripts for shock simulations
* MEAM potential files
* Representative Rankine–Hugoniot/CNA/dislocation datasets
* OVITO and Python post-processing scripts

## Software

* LAMMPS
* OVITO
* Python 
* Atomsk
